# 📚 LMS Web Application

## 📌 Introduction

This project is a Learning Management System (LMS) web application built using modern web technologies. It allows users to interact with learning content, manage sessions, and organize their learning experience efficiently.

The platform focuses on providing a smooth and responsive user experience with scalable architecture.

---

## ⚙️ Tech Stack

- **Next.js** – Full-stack React framework for building fast and scalable applications  
- **Supabase** – Backend-as-a-Service for database, authentication, and real-time features  
- **Tailwind CSS** – Utility-first CSS framework for modern UI design  
- **TypeScript** – Strongly typed JavaScript for better code quality  
- **shadcn/ui** – Pre-built UI components for clean and consistent design  
- **Zod** – Schema validation for ensuring data integrity  
- **Sentry** – Error tracking *(Not implemented yet)*  

---

## 🚀 Features

- 🔐 **User Authentication**  
  Secure login and signup system  

- 📚 **Learning Sessions**  
  Users can participate in and manage learning sessions  

- ⭐ **Bookmarks**  
  Save and revisit important content  

- 📜 **Session History**  
  Track previously accessed sessions  

- 🧩 **Reusable Components**  
  Modular and maintainable code structure  

- 📱 **Responsive Design**  
  Works smoothly across desktop and mobile devices  

- 🗄️ **Database Integration**  
  Real-time data handling using Supabase  

- 🎨 **Modern UI/UX**  
  Clean and user-friendly interface  

- 🔍 **Search Functionality**  
  Easily find relevant content
